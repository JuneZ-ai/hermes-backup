# 一房两门 · 用户术语体系

> 用户把 Windows 系统比喻成一间大房子，三个入口对应三种操作界面。

## 术语映射表

| 用户用语 | 标准术语 | 比喻 | 核心描述 |
|:--------|:--------|:----|:---------|
| **前门** / **GUI** | 图形用户界面 | 客厅·日常起居 | 点鼠标、看图标的 Windows 桌面环境 |
| **后门** / **CMD** | 命令提示符 / CLI | 维修通道·基础工具箱 | 那个黑色窗口，敲指令，工具箱里是螺丝刀锤子（dir/copy/del） |
| **总控台** / **总控界面** | PowerShell | 智能中控室·电动工具站 | 后门的升级版，能编程自动化、批量操控、远程控制 |

## 一句话区分

> 前门进去按开关看电视，后门进去检修电路拉电闸，总控台进去编程序定时开关全屋设备。

## 使用规则

- 用户可能混用口语和标准术语，都能接住
- 回应时优先使用用户的比喻体系，解释技术概念时保留标准术语
- 对话中自动适配：用户说"后门"→ CMD，说"总控台"→ PowerShell
