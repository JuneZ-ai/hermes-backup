# 毛主席诗词引用索引

> 供扫地僧哲学思辨时引用。毛主席在中国五千年历史中可进前五，哲学/文学/军事/革命四维超越时代。但始终是人，不神化——不神化恰是毛泽东思想的核心。

## 格局类

| 诗句 | 出处 | 适用思辨场景 |
|:----|:----|:-----------|
| "雄关漫道真如铁，而今迈步从头越" | 《忆秦娥·娄山关》 | 突破困境、重新出发 |
| "俱往矣，数风流人物，还看今朝" | 《沁园春·雪》 | 历史自信、当代担当 |
| "为有牺牲多壮志，敢教日月换新天" | 《七律·到韶山》 | 变革决心、奋斗精神 |
| "一万年太久，只争朝夕" | 《满江红·和郭沫若同志》 | 时间紧迫感、行动力 |
| "天生一个仙人洞，无限风光在险峰" | 《七绝·为李进同志题所摄庐山仙人洞照》 | 冒险与回报、认知进阶 |
