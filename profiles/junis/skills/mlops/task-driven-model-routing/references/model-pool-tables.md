# 模型池配置参考

> 当前模型池表（基于 2026-05 TokenHub 环境）
> 模型排名和可用性随云端平台更新而变化，请定期重新验证。

## 实测可用模型（TokenHub）

### 🏛️ 高价值任务主力池 (90~100)

| 模型 ID | 能力标签 | 最佳场景 | 限制 |
|:--------|:---------|:---------|:----|
| `kimi-k2.6` | 代码·智能体·长上下文·推理 | 复杂代码、长文档、多步骤智能体、研究推理 | 仅支持 temperature=1 |
| `glm-5` | 代码·智能体·工程交付 | 复杂工程、长周期智能体、高难度项目 | — |
| `minimax-m2.7` | 代码·办公·结构化 | 办公文档、PPT、项目方案、生产力 | — |

### 🎯 专项能力池 (85~89)

| 模型 ID | 能力标签 | 最佳场景 | 注意 |
|:--------|:---------|:---------|:----|
| `glm-5v-turbo` | 视觉·多模态 | UI截图→代码、设计稿分析、图文理解 | supportsImages=true |
| `glm-5-turbo` | 工具调用·自动化 | OpenClaw工具链、长链执行 | ⚠️ 需额外开通套餐 |

### ⚖️ 备选性价比池 (80~84)

| 模型 ID | 能力标签 | 最佳场景 |
|:--------|:---------|:---------|
| `deepseek-v4-flash` | 长上下文·低成本·高并发 | 超长文档、批量摘要、信息抽取 |
| `kimi-k2.5` | 多模态·开源·长文档 | 开源部署、私有化、视觉智能体 |
| `deepseek-v3.2` | 推理·搜索·工具调用 | 思考型任务、搜索型智能体、代码执行 |

### 🔬 探索成本补充池 (75~79)

| 模型 ID | 能力标签 | 最佳场景 |
|:--------|:---------|:---------|
| `hy3-preview` | 腾讯生态·试点 | 腾讯生态验证、原生适配、试点 |
| `minimax-m2.5` | 成本敏感·代码生成 | 批量代码生成、成本敏感重构 |

## 模型参数注意事项

| 模型 | 特殊参数限制 |
|:-----|:------------|
| `kimi-k2.6` | temperature 只接受 1，传其他值返回 400001 |
| `glm-5v-turbo` | supportsImages=true，视觉任务时在请求中可传 image_url |

## 验证命令

```bash
# 测试模型是否可用
curl -s -X POST 'https://tokenhub.tencentmaas.com/v1/chat/completions' \
  -H 'Authorization: Bearer <your-key>' \
  -H 'Content-Type: application/json' \
  -d '{
    "model": "<model-id>",
    "messages": [{"role":"user","content":"hi"}],
    "max_tokens": 10
  }'

# /v1/models 端点返回 404，无法通过 API 列举
# 只能逐模型测试
```
