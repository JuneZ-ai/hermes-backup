# 深度研究版（v2）笔记模板

> 用于将素材提炼为可深研的知识笔记。**「对普通人的启示」和「vs 其他思想家横向对比」是必选项，不可省略。**
>
> 已验证：段永平(519行)、曾鸣(481行)、黄奇帆(444行)、王阳明(305行)、邓小平(410行)、罗翔(430行)

## 章节结构（按顺序）

### 1. 人物定位（1-2节）
- 为什么这个人值得学
- 生平背景（关键节点）
- 在六大师框架中的角色

### 2. 思想体系展开（4-6根支柱）
每根支柱包含：
- **理论核心** — 一句话概括+展开
- **实战案例** — 具体事件佐证
- **警示信号** — 这个思想在什么条件下会失效/被滥用

### 3. 经典案例深度拆解
每个案例自成一节，结构：
- 背景 → 逻辑链（决策→行动→结果） → 启示

### 4. 「对普通人的启示」⭐ 必选
- 表格+清单+可操作步骤
- 用户最认可的部分
- 问自己：一个没有资源的普通人如何用这个思想？

### 5. vs 其他思想家的横向对比 ⭐ 必选
- 最少 3 个对照维度（表格呈现）
- 维度示例：思维层级、核心方法、关注焦点、对普通人的价值
- 参考 `six-masters-framework.md` 的对比表

### 6. 金句集（10-15条）
- 精华语录，每条带背景说明
- 选最震撼的，不要凑数

### 7. 在六韬知识库中的定位
- ASCII 图展示与其他模块/笔记的链接关系
- 至少 3 条发出链 + 3 条收到链

## 写作规范
- `version: v2.0` 标记
- 保持原文件名和路径（链接稳定性）
- 发出链 ≥ 20 条 `[[wiki链接]]`
- 收到链 ≥ 5 条（搜索文件名验证）
- 中文写作，术语保留英文

## 判定：v1 vs v2

| 素材类型 | 处理级别 | 示例 |
|---------|---------|------|
| 纯信息类（报告/数据/行业分析） | v1 标准入库 | 市场调研报告 |
| 思想/哲学/方法论（人物/传记/管理哲学） | **直接走 v2 深度** | 段永平、曾鸣、黄奇帆 |
| 历史/传记（以人物为核心的叙事） | 史鉴 v2 | 邓小平时代 |
| 法律/伦理/哲学 | 双螺旋 v2 | 罗翔 |

## 链接密度验证命令

```bash
# 统计发出链
grep -c "\[\[" "笔记路径.md"

# 统计收到链（排除自身）
grep -rn "笔记名称\|笔记别名" "/vault路径" --include="*.md" -c | grep -v "笔记自身" | wc -l
```
