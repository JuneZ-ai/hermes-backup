# 易哲体育赛事预测法（Yi-Zhe Sports Prediction）

## 适用场景

用六韬易哲（易经/梅花易数）结合体育分析，对大型赛事（世界杯/联赛/关键场次）做预测。产出作为知识库案例积累，不用于真实投注。

## 完整流程

### Step 1: 搜集赛事基本面

用 `curl` + 搜索引擎（Bing/搜狗）搜集：
- 对阵双方基本信息（国家队/俱乐部、FIFA排名/联赛排名）
- 历史交锋记录（特别是世界杯开幕战等关键场合的过往交手）
- 本届赛事形势（分组、出线形势、伤病、战术）
- 开幕战/淘汰赛的历史规律（东道主不败率等）

**网络限制应对**：国内环境可用 Bing（搜狗中文搜索更稳定）。FIFA 官网和维基百科通常不通。搜狗返回的搜索结果片段已经包含关键信息（如分组情况和历史渊源），不需要点进具体文章。

### Step 2: 梅花易数起卦

**起卦要素**

| 方法 | 说明 | 
|:----|:-----|
| 时间起卦（标准） | 用当前起卦时刻的农历年月日三数：上卦=(年+月+日) mod 8，下卦=(月+日+时) mod 8，动爻=(年+月+日+时) mod 6 |
| 时间起卦（简化） | 公历年(四位)+月+日 mod 8 即可，足够娱乐预测 |
| 数字起卦 | 用两队世界排名/队服颜色数等 |

**依赖**：`lunar-python` 包（`pip install lunar-python`）

```python
from lunar_python import Solar, Lunar
s = Solar.fromYmd(year, month, day)
l = s.getLunar()
# 获取完整八字、宜忌等信息用于解卦上下文
```

### Step 3: 卦象解读（体育赛事专用映射）

#### 64卦与比赛语义映射

| 占卜要素 | 体育赛事映射 |
|:---------|:------------|
| 体卦（下卦） | 主队/东道主/本队 |
| 用卦（上卦） | 客队/挑战者/对手 |
| 体用生克 | 体克用=主队有利；用克体=客队有利；体用比和=势均力敌 |
| 动爻 | 比赛转折点（动爻位置对应半场：初-二=开场，三-四=中段，五-上=收尾） |
| 变卦 | 最终趋势/结果走向 |
| 互卦 | 比赛过程中的隐藏因素（裁判/伤病/运气） |

**五行与球队风格的映射**：
- 乾金（刚健）→ 力量型/德国风格
- 兑金（锐利）→ 快速反击/葡萄牙风格
- 离火（炽热）→ 激情型/巴西风格
- 震雷（爆裂）→ 冲击型/英格兰风格
- 巽风（柔韧）→ 技术型/西班牙风格
- 坎水（流动）→ 掌控型/阿根廷风格
- 艮山（稳固）→ 防守型/意大利风格
- 坤土（厚重）→ 整体型/荷兰风格

#### 体用生克预测表

| 关系 | 比赛寓意 | 典型比分 |
|:----|:---------|:--------|
| 体克用（主克客） | 主队占优但不一定大胜 | 2-1, 1-0, 2-0 |
| 用克体（客克主） | 客队爆冷或防守反击奏效 | 0-1, 1-2 |
| 体用比和（同五行） | 势均力敌，可能平局 | 1-1, 2-2, 0-0 |
| 体生用（主生客） | 主队消耗大，场面占优但结果不佳 | 1-1, 2-2 |
| 用生体（客生主） | 客队送机会，主队轻松取胜 | 3-0, 3-1 |

### Step 4: 历史规律参照

结合历史数据验证卦象解读。世界杯开幕战的关键规律：

- **东道主开幕战不败率**极高（9届中8届不败，唯一例外是2022卡塔尔0-2厄瓜多尔——但卡塔尔属于亚洲杯冠军递补，非传统东道主）
- 开幕战通常**上半场节奏慢、比分小**（中孚卦"外柔"）
- **下半场后30分钟进球概率高**（变卦"风雷益"越打越开）
- **历史对阵重演倾向**：2010年墨西哥vs南非1-1→2026年可能重演类似剧情

### Step 5: 市场盘口验证（强制步骤）

**⚠️ 必须执行：卦象不能单独作为结论提交。** 缺少市场数据验证的预测是半成品。

#### 数据源

| 数据源 | 获取方式 | 说明 |
|:------|:---------|:-----|
| Oddsportal | `curl -sL --compressed` | 结构化JSON数据，含1X2赔率和让球盘 |
| 500.com/澳客网 | 搜狗搜索结果片段 | 国内可通，但属于JS渲染页需从搜索摘要中提取 |
| 搜狗微信 | `weixin.sogou.com` | 中文赛前分析文章中常含赔率数据 |

**环境注意**：Oddsportal 需要 `--compressed` 参数 + 合理的 UA。FIFA官网和维基百科通常不通，不要浪费token。

#### 读取1X2赔率 → 隐含概率

1X2 赔率直接用公式转为市场概率：
```
隐含概率 = (1 / 赔率) / (1/主胜 + 1/平局 + 1/客负) × 100%
```

赔率 < 2.00 → 市场认为该结果概率 > 50%（大热）
赔率 2.00-3.00 → 市场认为该结果概率 33%-50%
赔率 > 3.00 → 市场认为该结果概率 < 33%（冷门）

#### 读取亚洲让球盘 → 实力差判断

亚洲让球盘比1X2更能反映市场对「实力差」的真实判断：

| 盘口类型 | 含义 | 示例 |
|:---------|:-----|:-----|
| 平手(0) | 双方完全对等 | 澳门2.06 vs 1.80 = 客队不被看输 |
| 平/半(0.25) | 主队轻微优势但不能确信赢 | 188bet 2.49 vs 1.60 |
| 半球(0.5) | 主队被看好赢球 | 主队赔率1.80+ |
| 半/一(0.75) | 主队明显占优 | 主队赔率1.60+ |

#### 双轨对比框架

```
┌─────────────────┐     ┌─────────────────┐
│   易卦轨道       │     │   市场轨道       │
│  体用生克→方向   │     │  赔率盘口→概率   │
│  动爻爻辞→变数   │     │  趋势变化→信号   │
└────────┬────────┘     └────────┬────────┘
         └────────┬──────────────┘
                  ▼
          ┌─────────────────┐
          │  综合判断         │
          │  一致→高信心      │
          │  分歧→调低信心    │
          └─────────────────┘
```

**关键原则**：
- 卦象说"主有利"但赔率是"平手盘客队贴水低" → 不可忽略分歧，应调低主胜预期
- 卦象说"平局可能"且平局赔率在3.00附近 → 平局是高价值选项（赔率×概率 > 1）

### Step 6: 综合预测报告

最终报告应包括：
1. 起卦信息（时间/干支/卦象）
2. 本卦解读（卦辞+动爻爻辞的体育映射）
3. 体用生克分析（谁占优、什么程度）
4. 变卦趋势（比赛走势）
5. 历史参照（过往开幕战数据）
6. 综合预测（胜平负概率+最可能比分+一句话总结）

## 注意事项

- **仅供娱乐**：易哲预测本质是决策辅助/思维训练，不是确凿判断
- **卦象≠绝对胜负**：中孚卦未必赢球，益卦未必大胜——卦象提供的是**能量状态**而非**精确比分**
- **动爻是关键**：不看动爻就下结论，等于只看了比赛前五分钟
- **时间起卦的局限性**：同一时刻起的卦对所有赛事都是一样的，需要在解卦时做体育专用的映射和过滤
- **强烈推荐结合基本面分析**：卦象确定了能量方向（金克木），但具体比分（2-1还是1-0）需要基本面（历史交锋、伤病、战术、主场优势）来校准
