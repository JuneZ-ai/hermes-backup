# 六套核心思辨工具：五思一法 + 东方六器

## 五思一法

| 工具 | 本质 | 最适用场景（DNA-01映射） |
|------|------|--------------------------|
| **系统思维** | 洞察事物间的动态关联与整体规律 | 连接知识、判断全局影响（Step2） |
| **批判性思维** | 空杯心态甄别真伪，不盲从权威 | 筛选知识、验证信源（Step3） |
| **第一性原理** | 剥离表象直击本质，从公理推演 | 拆解框架、重构认知（Step1） |
| **逆向思维** | 反过来想，规避盲点，找突破口 | 风险排查、stop doing list |
| **成长思维** | 错误是优化知识库的养分 | 行为测试、三振晋升修复（Step4） |
| **合思法** | 双螺旋+多径原则+韬定律三问 | 复杂决策、跨模块深度分析 |

## 东方六器（思辨工具补充）

| 工具 | 来源 | 本质 | 和西方工具的关系 |
|------|------|------|-----------------|
| **体用如一** | 道家 | 透过现象求本质——"万变不离其宗" | 和第一性原理并行：一个拆要素，一个透视规律 |
| **阴阳转化** | 道家/易 | 顺境找隐患，逆境找转机 | 超越非黑即白，和理解动态系统互补 |
| **墨家三表法** | 墨家 | 历史经验+现实数据+实践效果三维验证 | 中国版的实证主义 |
| **中道思辨** | 佛学 | 离两边不呆滞，在动态中找平衡 | 超越辩证法的强制对立 |
| **诸行无常** | 佛学 | 一切现象刹那变化——消解路径依赖 | 对抗成长思维的"正面偏误" |
| **知止而后有定** | 儒家 | 先问终极目的再行动 | 以终为始的价值理性校准 |

## 按需查表

| 当需要… | 用这个工具 |
|---------|-----------|
| 拆解一个复杂概念 | 第一性原理 + 体用如一 |
| 判断新知识和旧框架的关系 | 系统思维 + 阴阳转化 |
| 验证一个观点的真伪 | 批判性思维 + 墨家三表法 |
| 测试新框架是否内化成功 | 成长思维 + 诸行无常 |
| 决定要不要归档/丢弃 | 逆向思维 + 为道日损 |
| 做复杂的两难决策 | 合思法 + 中道思辨 |
| 破除对过往成功的路径依赖 | 诸行无常 + 知止而后有定 |
