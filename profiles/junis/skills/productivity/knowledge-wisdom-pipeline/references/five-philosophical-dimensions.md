# 五向哲学：知识走向智慧的五个维度

知识升华为智慧不是技术问题，是哲学问题。每一维度都是一次"超越"。

| 维度 | 超越 | 核心洞见 |
|------|------|---------|
| **知→行** | 经验与实践的统一 | 知识是静态符号系统，智慧是动态创造性行动。王阳明："知是行之始，行是知之成。" |
| **特殊→普遍** | 本质的抽象与洞察 | 知识停留在具体现象（特殊），智慧穿透到底层规律（普遍）。亚里士多德：智慧即"第一原因和原理" |
| **求多→求少** | 消解与整合 | 老子："为学日益，为道日损。"追求知识做加法，追求智慧做减法 |
| **工具→价值** | 反思性的批判 | 知识回答"How"，智慧回答"Why"和"What is good" |
| **无知→觉知** | 主体的自我反省 | 苏格拉底："我唯一知道的，就是我一无所知。" |

**一句概括**：知识是主客体分离时主体对客体的理性建构；智慧是主客体融合后对存在本质的直觉体悟与德性实践。
