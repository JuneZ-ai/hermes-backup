# 搭建日志审计 SOP

> 扫描多维表格搭建日志（Build Log），找出遗漏、未完成和缺失记录。

## 触发条件

- 用户说"检查搭建日志"、"有没有遗漏"、"有没有未完成的工作"
- 定期巡检（如每周/隔周）
- 大批量操作后需要验证记录完整性

## 审计检查项（按优先级）

### 1. 空状态字段检查

查找「状态」字段为空的记录——这些是创建时未打标记的历史遗留。

**方法：** `+record-list` 全量导出 → 检查 `状态` 列是否为空。

**处理：**
- 如果是已完成的工作 → 更新状态为 `✅ 已完成`
- 如果确实未完成 → 更新为 `⏳ 待跟进`
- 如果是"今日无操作"占位记录 → 可标记为 `✅ 已完成`

### 2. 日期空白检查

按时间排序记录，查找是否存在日期断层（如最近几天的操作没记录）。

**方法：** 提取所有记录的 `时间` 字段，按日期排序，对比当前日期。

**处理：**
- 通过 session_search 回顾最近会话，确认是否有操作
- 如有未记录的操作 → 补录到搭建日志

### 3. 跨会话任务追溯

搜索近期会话（session_search），提取操作摘要，反向对比搭建日志：

```
会话摘要 → 核心操作列表
               ↓
          与搭建日志字段「操作内容」逐一匹配
               ↓
       未匹配项 = 缺失记录
```

**处理：**
- 有明确成果的 → 补录为 `✅ 已完成`
- 有进展但未闭环的 → 补录为 `⏳ 待跟进`
- 被用户要求但未执行的 → 补录为 `📋 计划中`

### 4. 状态一致性检查

**检查项：** 状态标记与备注是否矛盾。常见问题：
- 备注写"已完成"但状态为空 → 更新为 `✅ 已完成`
- 备注写"待验证"但状态为 `✅ 已完成` → 更新为 `⏳ 待跟进`
- 同一会话的多个操作状态应与实际执行情况一致

## 补录格式规范

新建记录字段：

| 字段 | 值 | 说明 |
|------|-----|------|
| 时间 | YYYY-MM-DD 00:00:00 | UTC+8，当天0点 |
| 操作内容 | 简明描述（含关键链接/编号） | 如有Obsidian链接用 `[[双链]]` |
| 领域 | 从现有选项选择 | 知识库基建 / API打通验证 / 工具配置 / 架构决策 / 飞书权限开通 |
| 状态 | ✅ 已完成 / ⏳ 待跟进 / 📋 计划中 | 按实际 |
| 沟通状态 | ✅ 顺畅 / ⏳ 卡壳 / ○ 无需沟通 | 按实际 |
| 备注 | 补充说明 | 如有归档编号（如`归档DS-22`） |

## 工作流规则（已固化到 memory）

> 每次在飞书创建文档/归档笔记/更新知识库等操作完成后，必须同步写入搭建日志，记录时间/操作内容/领域/状态/备注。用bot身份写入。

## 三表同步（搭建日志 + 每日记录 + 收藏随想录）

搭建日志审计完成后，必须同步检查并更新另外两张关联表——**不能只更新搭建日志**。

### 同步流程

```
搭建日志新增/补录完成
         ↓
   ✅ 每日记录：检查对应日期是否有记录
       → 没有：新增一条每日记录（公历时间/农历/节气/干支/记录内容/类型）
       → 已有：确认内容是否包含当天操作，不重复追加
         ↓
   ✅ 收藏随想录：本次操作涉及的新文章/资源/内容是否值得收藏
       → 有值得收藏的内容：新增收藏记录（标题/来源链接/感悟/来源类型/分类/评分/状态）
       → 没有可收藏内容：跳过
```

### 每日记录字段速查

| 字段 | 说明 | 示例 |
|------|------|------|
| 公历时间 | YYYY-MM-DD 00:00:00 | 2026-05-30 00:00:00 |
| 农历 | 月+日（如四月十四） | 四月十四 |
| 节气 | 当前节气 | 小满 |
| 干支 | 年柱 月柱 日柱 | 丙午年 癸巳月 甲辰日 |
| 记录内容 | 当天操作摘要 | Darwin skill升级、模型配置等 |
| 类型 | 单选：📚读书/💼工作/🤝会议/💡思考/🏠生活/🌐见闻/💻工具 | 💻 工具 |

### 收藏随想录字段速查

| 字段 | 说明 | 示例 |
|------|------|------|
| 标题 | 文章/资源标题 | 达尔文.skill v2.0升级 |
| 来源链接 | URL | https://mp.weixin.qq.com/s/... |
| 我的感悟 | 个人见解 | 花叔的达尔文Skill v2.0... |
| 金句摘录 | 原文金句（可选） |  |
| 来源类型 | 单选：📱微信文章/🌐网页/📚书籍/🎧播客等 | 📱 微信文章 |
| 分类 | 单选：💻技术AI/🏢商业管理/🧘个人成长等 | 💻 技术/AI |
| 收藏日期 | YYYY-MM-DD 00:00:00 | 2026-05-30 00:00:00 |
| 评分 | 单选：⭐~⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 状态 | 单选：🔖已收藏/📖在读/✅已读完/🔄待重温/✅已入库 | ✅ 已入库 |

### 日柱干支速算（Python）

```python
from datetime import date
TIANGAN = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸']
DIZHI = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥']
ref = date(2000, 1, 1)  # 2000-01-01 = 戊午日
target = date(2026, m, d)
diff = (target - ref).days
stem = (4 + diff) % 10  # 戊=idx4
branch = (6 + diff) % 12  # 午=idx6
day_ganzhi = TIANGAN[stem] + DIZHI[branch]
```
