# 三表同步工作流

> 适用范围：lark-cli base +record-upsert / +record-batch-update
> 使用时机：每次写入或修改搭建日志后

## 三表定位

| 表 | 表ID | 定位 | 写入频率 |
|----|------|------|----------|
| **搭建日志** | `tbllV9WgN64Zwput` | 每笔操作的原子记录 | 每次操作后 |
| **每日记录** | `tblTLllADiUdhL6e` | 每日工作总结+农历/节气/干支 | 每天至少1条 |
| **收藏随想录** | `tblydJHMALlK3stv` | 有保存价值的外部分享/文章/感悟 | 有内容时 |

## 铁则

**写搭建日志时，必须同步检查另外两张表是否需要更新。**

### 触发条件判断

```
写搭建日志后 → 问自己三个问题：

Q1: 这是今天第一条操作吗？
→ 是 → 检查每日记录是否有今天的条目：无则创建，有则补充内容
→ 否 → 跳过（今天的每日记录已有，但可追加补充）

Q2: 涉及外部内容（文章/链接/视频/书籍）吗？
→ 是 → 检查收藏随想录中是否已收录：无则创建（标题/来源链接/感悟/分类/评分/状态）
→ 否 → 跳过

Q3: 有需要用户知道的感悟/金句吗？
→ 是 → 补充到收藏随想录的对应字段（我的感悟/金句摘录）
→ 否 → 跳过
```

### 每日记录字段映射

| 字段名 | 取值规则 |
|--------|---------|
| 公历时间 | `YYYY-MM-DD 00:00:00`（当天0点） |
| 农历 | 从农历库计算（python lunardate 或 zhdate） |
| 节气 | 从太阳术语表匹配 |
| 干支 | 年柱+月柱+日柱（年柱固定，月柱看节气月份，日柱查表或计算） |
| 黄历宜忌 | 可选，留空也可 |
| 记录内容 | 当天所有操作摘要，用换行符分隔 |
| 类型 | 单选：📚读书 / 💼工作 / 🤝会议 / 💡思考 / 🏠生活 / 🌐见闻 / 💻工具 |

### 收藏随想录字段映射

| 字段名 | 取值规则 |
|--------|---------|
| 标题 | 原文标题或自定义标题 |
| 来源链接 | 原始URL，空也可 |
| 我的感悟 | 个人思考总结，2-5句 |
| 金句摘录 | 文中打动人的原句 |
| 来源类型 | 单选：📱微信文章 / 🌐网页 / 📚书籍 / 🎧播客 / 🎬影视 / 📰新闻 / 🎵歌曲 / 📨邮件通讯 / 💬聊天记录 / 📄PDF文档 / 🗂其他 |

| 收藏日期 | `YYYY-MM-DD 00:00:00` |
| 评分 | ⭐ / ⭐⭐ / ⭐⭐⭐ / ⭐⭐⭐⭐ / ⭐⭐⭐⭐⭐ |

### Python 农历计算参考

```python
import lunardate
from datetime import date

lunar = lunardate.LunarDate.fromSolarDate(2026, 5, 31)
print(f"{lunar.year}-{lunar.month}-{lunar.day}")
# → 2026-4-15  （四月十五）
```

干支日计算（以2000-01-01=戊午日为基准）：

```python
TIANGAN = ['甲','乙','丙','丁','戊','己','庚','辛','壬','癸']
DIZHI = ['子','丑','寅','卯','辰','巳','午','未','申','酉','戌','亥']
ref = date(2000, 1, 1)  # 戊午日
target = date(2026, 5, 31)
diff = (target - ref).days
stem = TIANGAN[(4 + diff) % 10]  # 戊=index 4
branch = DIZHI[(6 + diff) % 12]  # 午=index 6
print(f"{stem}{branch}")  # 乙巳
```

### 常见错误

- **只更新了搭建日志，忘了每日记录** → 用户会问"每日记录为什么没有同步"
- **收藏随想录字段不全** → 至少填 标题/来源链接/分类/收藏日期 四个字段
- **日期格式错误** → 使用 `YYYY-MM-DD 00:00:00` 格式，不要用 ISO 或相对时间
- **单选字段传值错误** → 传选项名称字符串，如 `"💻 工具"`，不是数组或ID
