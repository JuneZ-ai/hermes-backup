# MinerU/OCR Markdown 文本处理

> 当用户发送 MinerU OCR 工具导出的 markdown 文本时，需额外清洗和格式化。

## 典型特征

| 特征 | 示例 |
|:----|:-----|
| 嵌有 `<table><tr><td>` HTML 表格 | OCR 工具用 HTML 表格排版多列内容 |
| 合并行（两行诗被拼成一行） | `"春江潮水连海平海上明月共潮生"`（本应为两句） |
| 全角 variant 字符 | `「`、`》`、`⻁`（而非标准 `「`、`》`、`虎`） |
| 章节标题与正文无分隔 | `"七律 到韶山一九五九年六月别梦依稀咒逝川"`（标题、年代、正文连在一起） |
| 脚注/注释格式混乱 | 注释放置不规则或混入正文 |

## 通用处理流程

### Step 1：检测并提取 HTML 表格

```bash
grep -oP '<table>.*?</table>' input.md | head -1
```

如果存在 HTML 表格：注意，MinerU 的 HTML 表格使用 `rowspan` 属性表示跨行归属，需要按 `rowspan` 分组还原。

**关键处理模式：**

```
<table>
  <tr><td rowspan="8">诗A标题</td><td>诗A正文行1</td></tr>
  <tr><td>诗A正文行2</td></tr>
  <tr><td>诗B标题</td></tr>
  <tr><td>诗B注释</td></tr>
  <tr><td rowspan="3">诗C标题</td><td>诗C正文行1</td></tr>
  <tr><td>诗C正文行2</td></tr>
  <tr><td>诗C正文行3</td></tr>
</table>
```

**处理规则：**
1. `rowspan=N` 的第一个 `<td>` 是标题/归属列，其后 N 行都属于这个标题
2. `rowspan` 结束后（N行用完），下一个 `<tr>` 的标题列变为新条目
3. 同一 `<tr>` 中可能有纯文本注释（无标题列），属于上一首诗的附注
4. 提取后按「标题 + 年代 + 序言（如有）」→「正文」→「附注/和诗」三段式重建

### Step 2：分离合并的标题+年代+正文

OCR 常把一首诗的标题、年代、正文连成一行，如：

```
七律 到韶山一九五九年六月二十五日到韶山,离别这个地方已有三十二周年了别梦依稀咒逝川 故园三十二年前
```

恢复为三段式：

```markdown
### 七律·到韶山（1959年6月）

> 一九五九年六月二十五日到韶山。离别这个地方已有三十二周年了。

别梦依稀咒逝川，故园三十二年前。
红旗卷起农奴戟，黑手高悬霸主鞭。
...
```

**判断规则：**
- 诗体（七律/五律/七绝/沁园春/满江红等）+ 年代 → 标题行
- `一九` + `年`/`月` → 年代
- 带 `,` 的较长描述 → 序言/题注（用 blockquote 包裹）
- 其他 → 正文，按韵律/字数断行

### Step 3：诗歌正文断行

OCR 会丢失原文换行。按诗体规则恢复断行：

| 诗体 | 每句字数 | 每首句数 | 示例断行 |
|:----|:--------|:--------|:--------|
| 五绝 | 5字 | 4句 | `白日依山尽，黄河入海流。` |
| 五律 | 5字 | 8句 | `国破山河在，城春草木深。` |
| 七绝 | 7字 | 4句 | `朝辞白帝彩云间，千里江陵一日还。` |
| 七律 | 7字 | 8句 | `红军不怕远征难，万水千山只等闲。` |
| 词 | 变长 | 变数 | 按词牌格律分行（如沁园春的上片下片） |
| 古风/杂言 | 不定 | 不定 | 按韵脚/语义断行 |

**辅助判断：** 用句式平行、对仗结构、韵脚来识别句尾。

### Step 4：添加 Frontmatter 和元数据

```yaml
---
tags:
  - 六韬/六韬易哲
  - 文史哲
  - 作者名
  - 诗词/文学
author: 作者
---
```

### Step 5：年代分段

对于全集类作品（如诗集、文集），按年代/时期分节：

```markdown
## 1910年代

### 诗题（1910年）
...

## 1920年代

### 诗题（1920年）
...
```

### Step 6：附注/序言处理

- 短序言（百字内）→ 用 `> blockquote` 放在诗题下方、正文上方
- 长注释 → 放在全诗的末尾，用 `**注：**` 标记
- 附注中的他人唱和作品 → 用 `**附：**` 标记，保持与主作品分隔

## 实战案例：毛诗全集清洗

见 2026-05-29 会话：MinerU 版《毛泽东诗词全集》处理

| 问题 | 处理方式 |
|:----|:---------|
| HTML `<table>` 内含8首诗+注释 | 逐一提取 `<td>` → 重构为独立诗条目 |
| 七律/七绝正文合并成一行 | 按7字一句断行（七律=8行，七绝=4行） |
| 年代与正文粘连 | 识别 `一九` + 数字 → 分离出年代行 |
| 长序言混入正文 | 用 `> blockquote` 包裹 |
| 附注原诗 | `**附：**` 标记，独立成段 |
