## Reading Method per Node

For each node, use 三步走:
1. **读核心事件** — what happened (1-2 key figures, 1-2 key decisions)
2. **对应六韬易哲问自己:**
   - 韬: what strategic pattern is at play?
   - 易: what is the transformational moment?
   - 哲: what principle does this reveal about human nature?
3. **写一句话读通** — one sentence that distills the entire node

## Example: 三家分晋 (Node 1)

**核心:**
- 智伯才胜德 → 水灌晋阳 → 韩魏反水 → 智氏灭族
- 赵襄子守晋阳三年不降

**六韬易哲 mapping:**
- 韬: 智伯赢了每一步但输了全局。赵襄子选对位置然后熬。
- 易: 周天子封韩赵魏为诸侯—"礼"(旧规则)向"力"(新规则)的相变点。
- 哲: 司马光问"才与德孰重"——但真正的答案是:才和德都是工具,关键在于"守仁"(知道什么不能做,什么必须分)。

**一句话读通:**
> 智伯的才是一把往外打的拳,越打越累;赵襄子的德是往里收的桩,越站越稳。
