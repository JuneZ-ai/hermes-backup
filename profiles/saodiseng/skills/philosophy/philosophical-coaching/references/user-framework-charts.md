## Figure 2: 时间管理 · 内在秩序 (Time Management · Inner Order)

### 时间自由公式
充裕时间 = (外部极简 + 内部有序) x 单点聚焦

### 根源洞察
- 混乱模式: 外部堆积、内部内耗、多任务切换 → 忙/焦/没效率
- 有序模式: 外部清爽、内部目标清晰、单核处理 → 松/快/有成果

### 外部秩序
- 物理环境: 桌面只留当前任务相关
- 数字环境: 关闭通知，定期查收
- SOP化: 重复事情建立流程清单

### 内部秩序
1. 清空大脑缓存 (GTD)
2. 极度聚焦 (MIT)

### 单核工作循环
选择一件事 → 屏蔽一切干扰 → 深潜25分钟专注 → 休息

### 能量匹配
- 高能时刻: 早起 / 深度工作
- 低能时刻: 午饭后 / 疲惫时
- 混乱时刻: 焦虑 / 被打断时

### 立刻整理的5个微行动
1. 花2分钟整理桌面
2. 把脑子里担心的事写下来
3. 关闭手机非必要通知
4. 选出当天必须完成1件事
5. 每工作45分钟，休息5分钟
