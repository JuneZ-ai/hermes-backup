# User's Ancestral Reference: 左宗棠

Used as philosophical touchstone in coaching conversations.

## Three Details That Worked

1. **"身无半亩，心忧天下 / 读破万卷，神交古人"** — The user's motto "知行守仁、以术载道" traces to this 左公 spirit: cultivate real knowledge (实学: 舆地、农政、兵事), not empty talk.

2. **63岁抬棺出关收复新疆** — "发上等愿" in action, not waiting for conditions to be perfect. Key detail: left 李鸿章 arguing "新疆不毛之地, 不如买海军". Left's counter: "重新疆者，所以保蒙古；保蒙古者，所以卫京师."

3. **左公柳** — From 潼关 to 哈密, planted 杨柳三千里. No budget, did it anyway. Action creates the path.
